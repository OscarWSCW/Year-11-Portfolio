Hangman

Overview

Our project is a more simple Hangman game which is aimed to be easier for kids to play.

End Users

our targeted audience for our game is mainly kids who are aged around 5 to 12, we've decied to make the game more easier by aiming for easier words and topics so the game is easier for kids. we also decied to show the rules and a tutorial before startihng the game just in case if the kids don't know how to play.

Requirements

Introduction

    The introduction will tell you before you start the game there will a introduction and tutorial that introduces people to the game and shows them how to play.

Win/Loss Checker

    The win/loss checker will tell the user during the game if they guess a letter correctly or not. it will also help the user on what to pick next if the user got the question wrong.

Conclusion

    The conclusion will tell the user if the game has ended or not.
